Starting a blog on GitHub-Pages with Hexo

Build a blog (like this one) on GitHub-Pages in three simple steps!

Step 1. Setup GitHub-Pages and Hexo

Create GitHub-Pages

GitHub Pages is a static site hosting service for GitHub users. Follow the steps on the GitHiub-Pages website to create a user site for yourself. It should be accessable at https://<your-username>.github.io.

Install Hexo

Hexo is a markdown-based blog framework that powered this blog. Follow the Hexo Installation Doc to install Hexo on your computer. You may need to install the requirements (Node.js and Git) first if you do not have them.
